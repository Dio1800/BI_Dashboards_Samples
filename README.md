# BI_Dashboards_Samples
Public portfolio with samples of business intelligence dashboards and reports

MySQL Sakila sample database
The dashboards and reports where build using data from MySQL Sakila sample database available on
https://dev.mysql.com/doc/index-other.html.
The files were also included on this repository to make sure this specific version will always available.

Power BI Desktop Dashboard
The first sample was developed on Power BI Desktop tool

Other tools may be used in future. One folder will be created for each additional BI Tool.
